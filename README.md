# Simple Banking System

## Overview

The Simple Banking System is a Python-based console application designed to simulate basic banking operations. It enables users to perform common financial transactions such as depositing money, withdrawing money, and checking their account balance through a simple terminal interface.

The project demonstrates the use of structured programming concepts while providing a practical solution to basic financial record management.

---

## Problem Statement

Many individuals and small business owners still manage their finances manually using notebooks or paper records. This approach can lead to:

- Calculation errors
- Loss of financial records
- Incorrect account balances
- Time-consuming manual calculations
- Poor financial decision-making

This application addresses these challenges by automating basic banking operations, making financial management faster, more accurate, and more reliable.

---

## Features

- Deposit money into an account
- Withdraw money from an account
- Check current account balance
- Prevent withdrawals that exceed available balance
- Display appropriate error messages for invalid transactions
- Simple and user-friendly terminal interface

## How to Run

1. Install Python 3 if it is not already installed.
2. Download or clone this repository.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the program:

```bash
python banking_system.py
```

## Programming Concepts Used

This project demonstrates several structured programming principles, including:

- **Sequence** – Instructions execute in a logical order.
- **Selection** – Uses `if`, `elif`, and `else` statements for decision-making.
- **Modularity** – Banking operations are divided into separate functions.

---

## Validation

The system validates user input by:

- Preventing withdrawals that exceed the available balance.
- Displaying an error message for invalid transactions.
- Maintaining an accurate account balance after each transaction.
